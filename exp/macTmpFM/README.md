# Mongoose Wizard Project

This project is generated by Mongoose Wizard, https://mongoose.ws/wizard/
Mongoose Wizard is a visual Web dashboard builder for embedded devices.

This is a Web Filemanager app writen using the Mongoose UI. It is written to compile on Linux or Mac and allows files in the "/tmp" folder to be managed. 

In main.c are function to use the POSIX file functions to impliment filemanagement.
