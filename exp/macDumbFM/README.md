# Mongoose Wizard Project

This project is generated by Mongoose Wizard, https://mongoose.ws/wizard/
Mongoose Wizard is a visual Web dashboard builder for embedded devices.

The projects mocks up a File Manager Web UI. Then demonstrates how the glue functions can be rewritten as "my" functions in the main.c file.

This mock up was written on my mac rather than a Pico 2 W to speed up development.  Mongoose lets you write the same code everywhere!
